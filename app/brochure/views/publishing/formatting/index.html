Formatting

You can use a variety of standard tools to add formatting to your posts. Markdown to add links, lists and much more. Blot also renders mathematics and provides highlighting for code.

Guides

  • Markdown
    A lightweight markup language with plain text formatting syntax.
  • \(\TeX\)
    A math typesetting library with powerful formatting features.
  • Layout
    A custom Markdown extension which offers simple layout directives.
